How to run the code after git clone:
    
    Open command prompt

    1.) cd tab to the folder with app.py
    
    2.) set FLASK_APP = app.py
    
    3.) set FLASK_DEBUG=1
    
    4.) flask run (if this doesn't work use the command below)
        python -m flask run
